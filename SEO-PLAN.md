```
├── cases/                        # 新增：案例展示

├── news/                         # 新增：新闻资讯
│   └── index.html
├── faq/                          # 新增：常见问题
│   └── index.html
├── about/
└── contact/
```

### 3.2 技术优化项
- [ ] 创建 sitemap.xml
- [ ] 创建 robots.txt
- [ ] 添加结构化数据（Schema.org）
  - Organization
  - Product
  - LocalBusiness
- [ ] 优化页面加载速度
  - 图片压缩
  - CSS/JS压缩
  - 懒加载
- [ ] 移动端适配检查
- [ ] 添加Canonical标签

### 3.3 结构化数据示例
```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "当康T320健康一体机",
  "description": "旗舰款健康一体机，21.5英寸大屏...",
  "brand": {
    "@type": "Brand",
    "name": "当康健康"
  },
  "offers": {
    "@type": "Offer",
    "availability": "https://schema.org/InStock"
  }
}
```
